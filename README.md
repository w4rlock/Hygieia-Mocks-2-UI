# Hygieia-Mocks-2-UI

## REQUIREMENTS:
- docker
- docker-compose
- node 10.9.0
- npm 

## INSTALLATION

```bash
terminal$ git clone https://github.com/w4rlock/Hygieia-Mocks-2-UI.git hygieia
terminal$ cd hygieia
terminal$ ./setup.sh
terminal$ ./start-infra.sh
```


## NEW UI
```bash
terminal$ cd hygieia 
terminal$ launch-new-ui.sh

## OLD UI
```bash
terminal$ cd hygieia 
terminal$ launch-old-ui.sh
```
